## Honest interpretation

Across 20 (corpus, feature) Mann-Whitney U tests, 0 showed p<0.05.

At multiple-testing-naive α=0.05, we'd expect ~5% false positives under the null hypothesis. With 20 tests, that's ~1.0 expected false positives by chance. **0 significant findings is at or below chance** — H14's second premise NOT supported by this data.

## Caveats

- Function-name matching against .ANSWERS.md is heuristic (camelCase regex)
- N=6 corpora is still small (function-level N is larger but stratified by corpus)
- Mann-Whitney U is non-parametric but bug-class sample sizes vary widely across corpora
- No multiple-testing correction applied; with Bonferroni at α=0.05 / {total_tests} threshold would be much stricter
- Call graph is one representation; data-flow / type / inheritance graphs might show different signal
- Tree-sitter call extraction misses dynamic dispatch and library calls
- Even strong signal here doesn't reach 'almost trivially solve auditing' — it would suggest geometry is a useful PRIOR
